Html Form Elements

Form Element Description Example
<form> Container for form elements <form>...</form>
<input> Input field for user data <input type="text">
<label> Label for input fields <label>Name:</label>
<button> Clickable button <button>Submit</button>

HTML Table Elements

Table Element Description Example
<table> Defines a table <table>...</table>
<tr> Defines a table row <tr>...</tr>
<th> Defines a header cell <th>Header</th>
<td> Defines a table cell <td>Data</td>
<thead> Groups header content <thead>...</thead>
<tbody> Groups body content <tbody>...</tbody>
<tfoot> Groups footer content <tfoot>...</tfoot>

HTML Elements for Displaying Text

Element Description Example
<div> Defines a division or section <div>Content</div>
<span> Inline container for text <span>Text</span>
<p> Defines a paragraph <p>This is a paragraph.</p>
<h1> to <h6> Defines HTML headings <h1>Heading 1</h1>
<br> Inserts a single line break <br>
<hr> Inserts a thematic break (horizontal rule) <hr>
