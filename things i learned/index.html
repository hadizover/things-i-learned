HTML Table Elements

Table Element Description Example
<table> Defines a table <table>...</table>
<tr> Defines a table row <tr>...</tr>
<th> Defines a header cell <th>Header</th>
<td> Defines a table cell <td>Data</td>
<thead> Groups header content <thead>...</thead>
<tbody> Groups body content <tbody>...</tbody>
<tfoot> Groups footer content <tfoot>...</tfoot>

Learning HTML: Displaying < and >

while i was searching how to make this site i learned something new In HTML, if i wanted to show the less-than sign < and the greater-than sign > as text (not as tags, or code), i must use &lt; for < and &gt; for >.
For example:
&lt;form&gt; will display as <form>.
